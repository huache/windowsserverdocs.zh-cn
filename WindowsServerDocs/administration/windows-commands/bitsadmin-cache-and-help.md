显示 **缓存** 开关的命令行用法。

## <a name="syntax"></a>语法

```
bitsadmin /cache /help
```

## <a name="examples"></a>示例

显示 **缓存** 开关的命令行帮助。

```
bitsadmin /cache /help
```
