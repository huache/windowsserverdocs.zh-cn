# <a name="rsh"></a>rsh

在运行 RSH 服务或后台程序的远程计算机上运行命令。

> [!IMPORTANT]
> **Rexec**命令已弃用，因此在 Windows 中不能保证受支持。

安装基于 UNIX 的应用程序的子系统后，可以打开 C Shell 或 Korn Shell 命令窗口并运行 **rsh**。 有关详细信息，请在 C Shell 或 Korn Shell 提示符下键入 **man rsh** 。
