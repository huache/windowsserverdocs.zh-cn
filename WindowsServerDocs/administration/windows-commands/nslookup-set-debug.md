# <a name="nslookup-set-debug"></a>nslookup set debug

启用或禁用调试模式。

## <a name="syntax"></a>语法

```
set [no]debug
```

### <a name="parameters"></a>参数

| 参数 | 说明 |
| ---------- | ---------- |
| nodebug | 关闭调试模式。 这是默认值。 |
| debug | 启用调试模式。 启用调试模式后，可以查看有关发送到服务器的数据包的详细信息和生成的答案。 |
| /? | 在命令提示符下显示帮助。 |
| /help | 在命令提示符下显示帮助。 |
