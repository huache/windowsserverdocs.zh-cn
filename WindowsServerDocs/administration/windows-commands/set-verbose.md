# <a name="set-verbose"></a>设置详细

指定是否在创建卷影副本期间提供详细输出。 如果不使用参数，则 **将 "设置详细** 信息" 显示在命令提示符下。

## <a name="syntax"></a>语法

```
set verbose {on | off}
```

### <a name="parameters"></a>参数

| 参数 | 说明 |
|-----------|-------------|
|    {on    |    off}     |

## <a name="remarks"></a>备注

-   如果详细模式为 on，则 **set** 提供有关编写器包含或排除的详细信息，以及元数据压缩和提取的详细信息。
