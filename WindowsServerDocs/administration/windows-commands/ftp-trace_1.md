# <a name="ftp-trace"></a>ftp trace

> 适用于： Windows Server (半年通道) ，Windows Server 2019，Windows Server 2016，Windows Server 2012 R2，Windows Server 2012

切换数据包跟踪。 此命令还显示运行 **ftp** 命令时的内部 FTP 函数调用序列。

## <a name="syntax"></a>语法

```
trace
```

### <a name="examples"></a>示例

打开和关闭跟踪，请键入：

```
trace
```
