# <a name="autofmt"></a>autofmt

启动自动文件系统格式实用程序，该实用程序在从 Windows 恢复控制台调用时格式化驱动器或分区。

> [!IMPORTANT]
> 不能直接从命令行运行 **autofmt** 。
