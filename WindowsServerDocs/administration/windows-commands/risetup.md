# <a name="risetup"></a>risetup

通过从原始 Windows Server 2003 CD 或包含这些文件的自定义分发文件夹拉取文件来创建操作系统映像。

> [!IMPORTANT]
> **Rexec**命令已弃用，因此在 Windows 中不能保证受支持。
