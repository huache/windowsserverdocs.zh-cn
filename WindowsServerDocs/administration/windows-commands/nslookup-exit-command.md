# <a name="nslookup-exit"></a>nslookup/exit

退出 nslookup 命令行工具。

## <a name="syntax"></a>语法

```
nslookup /exit
```

### <a name="parameters"></a>参数

| 参数 | 说明 |
| --------- | ----------- |
| /? | 在命令提示符下显示帮助。 |
| /help | 在命令提示符下显示帮助。 |
