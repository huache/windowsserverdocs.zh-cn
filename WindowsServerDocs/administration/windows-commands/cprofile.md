# <a name="cprofile"></a>cprofile

Cprofile 已被弃用，并且在将来的 Windows 版本中不保证受到支持。

终端服务已更新为远程桌面服务。 有关详细信息，请参阅[欢迎使用远程桌面服务](../../remote/remote-desktop-services/welcome-to-rds.md)。
