# <a name="nslookup-set-d2"></a>nslookup set d2

打开或关闭详细调试模式。 打印每个数据包的所有字段。

## <a name="syntax"></a>语法

```
set [no]d2
```

### <a name="parameters"></a>参数

| 参数 | 说明 |
| ---------- | ---------- |
| nod2 | 关闭详细调试模式。 这是默认值。 |
| d2 | 启用详细调试模式。 |
| /? | 在命令提示符下显示帮助。 |
| /help | 在命令提示符下显示帮助。 |
