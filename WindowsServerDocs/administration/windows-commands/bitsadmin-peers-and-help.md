# <a name="bitsadmin-peers-and-help"></a>bitsadmin peers 和 help

显示 **/peers** 开关的命令行用法。

## <a name="syntax"></a>语法

```
bitsadmin /peers /help
```

## <a name="examples"></a>示例

显示 **/peers** 开关的命令行用法：

```
bitsadmin /peers /help
```
