# <a name="rexec"></a>rexec

在远程主机上运行指定的命令。 远程主机必须运行 rexecd service (或 daemon) ，才能连接到。

> [!IMPORTANT]
> **Rexec**命令已弃用，因此在 Windows 中不能保证受支持。
