# <a name="detail-volume"></a>detail volume

显示当前卷所在的磁盘。 在开始之前，你必须选择一个卷才能使此操作成功。 使用 " [选择音量](select-volume.md) " 命令选择卷并将焦点移动到该卷。 卷详细信息不适用于只读卷，如 DVD-ROM 或 CD-ROM 驱动器。

## <a name="syntax"></a>语法

```
detail volume
```

## <a name="examples"></a>示例

若要查看当前卷所在的所有磁盘，请键入：

```
detail volume
```
