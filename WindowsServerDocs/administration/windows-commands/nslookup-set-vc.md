# <a name="nslookup-set-vc"></a>nslookup set vc

指定将请求发送到服务器时是否使用虚拟线路。

## <a name="syntax"></a>语法

```
set [no]vc
```

### <a name="parameters"></a>参数


| 参数 | 说明 |
| ---------- | ---------- |
| novc | 指定在向服务器发送请求时从不使用虚拟线路。 这是默认值。 |
| vc | 指定在向服务器发送请求时始终使用虚拟线路。 |
| /? | 在命令提示符下显示帮助。 |
| /help | 在命令提示符下显示帮助。 |
