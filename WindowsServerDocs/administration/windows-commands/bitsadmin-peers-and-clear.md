# <a name="bitsadmin-peers-and-clear"></a>bitsadmin peers 和 clear

清除对等列表。

## <a name="syntax"></a>语法

```
bitsadmin /peers /clear
```

## <a name="examples"></a>示例

清除对等列表。

```
bitsadmin /peers /clear
```
