# <a name="bitsadmin-peers-and-list"></a>bitsadmin peers 和 list

列出所有对等方。

## <a name="syntax"></a>语法

```
bitsadmin /peers /list
```

## <a name="examples"></a>示例

列出所有对等方：

```
bitsadmin /peers /list
```
