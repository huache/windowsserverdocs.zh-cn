# <a name="best-practices-for-securing-active-directory"></a>保护 Active Directory 的最佳方案

>适用于：Windows Server 2016、Windows Server 2012 R2、Windows Server 2012

本文档提供实践者的视角，并包含一套实用的技术，可帮助 IT 管理人员保护企业 Active Directory 环境。 Active Directory 在 IT 基础结构中扮演关键角色，并确保不同网络资源在全球相互连接的环境中的协调性和安全性。 所讨论的方法很大程度上基于 Microsoft 信息安全和风险管理 (ISRM) 组织的经验，该经验负责保护 Microsoft IT 及其他 Microsoft 业务部门的资产，并向所选数量的 Microsoft 全球500客户提供建议。
